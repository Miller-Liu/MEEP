## Features

- **LLM-Powered Responses**: Utilizes large language models to generate intelligent, context-aware replies.
- **Custom Notion Integrations**: Connects seamlessly with Notion's API to read, write, and interact with your workspace.
- **Efficient Text-Based Integrations**: Simple and intuitive texting-style integrations designed for low-latency, low-cost operation.

## System Requirements

Before getting started, ensure you have the following installed on your system:

- **Pip**: Package management system
- **Pyenv**: Python version management


## Getting Started
### 1. Clone the repository
```bash
git clone <repository-url> <your-repo-name>
cd <your-repo-name>
```
### 2. Set up the virtual environment 
First, set the local Python version to 3.11.9.
```bash
pyenv local 3.11.9
```
To double check that the local version is 3.11.9, run
```bash
pyenv local # should be 3.11.9
```
Create the virtual environment
```bash
python -m venv .venv
source .venv/bin/activate
```
Install dependencies
```bash
pip install -r requirements.txt
```
